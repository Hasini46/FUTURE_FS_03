# 🏪 Local Business Website — Ember & Bean Café

**Task 3 | Future Interns — Full Stack Web Development**

## 📌 Project Overview
A professional, fully responsive website built for a local café business as part of the Future Interns internship program.

## 🔗 Live Demo
👉 [View Live Website](https://hasini46.github.io/FUTURE_FS_03)

## 🛠️ Tools Used
- HTML5
- CSS3 (Animations, Flexbox, Grid)
- Vanilla JavaScript

## ✅ Features
- Responsive navigation with mobile menu
- Hero section with CTA buttons
- Animated marquee strip
- About section with stats
- Filterable menu (Coffee / Food / Cold)
- Photo gallery with hover effects
- Customer testimonials
- Hours & contact section
- Table reservation form
- Fully mobile responsive

## 📢 Pitch Summary
This website helps the business grow by improving online discoverability, creating a strong first impression, and allowing customers to view the menu and reserve tables — all in one place.
